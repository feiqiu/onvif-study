这里的代码是《ONVIF协议网络摄像机（IPC）客户端程序开发》专栏文章的示例代码，方便大家学习ONVIF：


【示例代码包括】

discovery     - 设备搜索
deviceinfo    - 获取设备基本信息
systemtime    - 设备校时
capabilities  - 设备能力
avstream      - 读取音视频流
snapshot      - 图像抓拍
SetResolution - 修改分辨率



【其他说明】

「ffmpeg-linux-pc」和「openssl-linux-pc」是在我个人的linux环境下生成的库文件，不适合大家的Linux环境，你们需要自己编译这两个库
